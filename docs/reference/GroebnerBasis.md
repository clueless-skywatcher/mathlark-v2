## `GroebnerBasis`

Computes the reduced Groebner basis of a polynomial ideal using Buchberger's algorithm.

### Signature

```
GroebnerBasis(polys, vars)
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `polys` | List of PolynomialExpression | The generating polynomials of the ideal |
| `vars` | List of PolynomialExpression | The variables, which also determine the monomial ordering |

### Description

Given a list of polynomials and a list of variables, `GroebnerBasis` computes the unique reduced Groebner basis for the ideal generated by those polynomials under lexicographic monomial ordering.

The algorithm proceeds in two phases:

**Phase A: Buchberger's Algorithm**

Starting from the input generators, the algorithm iteratively computes S-polynomials for each pair of basis elements and reduces them modulo the current basis. If a reduction yields a nonzero remainder, it is added to the basis. This repeats until no new elements are produced.

**Phase B: Reduction**

The raw Groebner basis is then reduced in three steps:

1. **Minimize** — Remove any element whose leading monomial is divisible by the leading monomial of another element.
2. **Make monic** — Divide each element by its leading coefficient so that all leading coefficients are 1.
3. **Inter-reduce** — Reduce each element modulo the remaining elements, replacing or removing it as needed, until no further changes occur.

The result is the unique reduced Groebner basis for the given ideal and ordering.

### Examples

```
x := Var('x')
y := Var('y')
z := Var('z')

GroebnerBasis([x^2 - 1], [x])
=> [x^2 - 1]

GroebnerBasis([x + y, y + 1], [x, y])
=> [x - 1, y + 1]

GroebnerBasis([x^2 + y, x*y - 1], [x, y])
=> [x + y^2, y^3 + 1]

GroebnerBasis([x + 2*y - 1, x^2 + 2*y^2 - x], [x, y])
=> [x + 2y - 1, y^2 - (1/3)y]

GroebnerBasis([x + y + z, x*y + y*z, x*y*z - 1], [x, y, z])
=> [1]
```

### Errors

- Throws `WrongParameterLengthException` if not called with exactly 2 arguments.
- Throws `WrongParameterTypeException` if either argument is not a list, or if any element of the lists is not a `PolynomialExpression`.
